\section{Introduction}

Software engineering is a field of computer science dealing with software systems. Programming skills are not enough:

\begin{itemize}
	\item A \textit{programmer} develops a complete program, working individually (alone) on known specifications;
	\item A \textit{software engineer}, instead, identifies requirements and develops specifications, designing a component to be combined with other components, developed, maintained, used by others. This component can become part of several systems. Most importantly, a software engineer works in a team.
\end{itemize}

Initially, there was no reference model. As a reaction to the many problem, it was born the traditional \textit{waterfall} model and, later, flexible processes: \textit{iterative models}, \textit{agile methods}, \textit{DevOps}.
