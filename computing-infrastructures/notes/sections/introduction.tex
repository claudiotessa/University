\section{Introduction}

\subsection{Data centers}

A data center is recognized as a facility used to house computer systems and associated components.
it is a centralized physical or virtual repository designed for the storage, management, and dissemination of information pertaining to a particular business.

Data centers require a complex combination of specialized equipment.
This can be broadly categorized into two main groups:

\begin{itemize}
	\item \textbf{IT equipment}: represents the computing core of the facility.
	      Includes multiple servers with dedicated data storage systems and complex network infrastructures.
	\item \textbf{Facility equipment}: to support the IT hardware, the building itself must act as a highly secure site: redundant electrical systems, redundant climate control systems, etc.
\end{itemize}
