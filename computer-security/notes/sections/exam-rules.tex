\section*{Exam rules}

Exam structure:
\begin{itemize}
	\item Written test (up to 34 points): theory and practical exercises. Old exams previous to 2021-2022 have a different structure than the current exam. Closed book, no remote, \textit{no oral exam}.
	\item Optional homework challenges (up to 3 points): 6 challenges to be solved in the last 2 weeks of may. Grades are based on how many people pass the challenge.
	      \begin{itemize}
		      \item Week 1: memory errors vulnerabilities (buffer overflow and format string);
		      \item Week 2: web vulnerabilities (client + server)
	      \end{itemize}
\end{itemize}
